Installing Raspberry Pi OS

Beginner-friendly: This section guides you through installing Raspberry Pi OS using Raspberry Pi Imager.

Compatible Devices

  • Raspberry Pi 5
  • Raspberry Pi 4
  • Raspberry Pi 3
  • Raspberry Pi Zero 2 W (not fully tested)

Step-by-Step: Using Raspberry Pi Imager

  1. Open Raspberry Pi Imager
    Download from https://www.raspberrypi.com/software/ on your computer.
  2. Choose device
    Select your Raspberry Pi model.
  3. Choose operating system
    Raspberry Pi OS (Other) → Raspberry Pi OS Lite (64-bit)
  4. Choose storage
    Select your SD card or USB drive.
  5. OS customization settings
    • Set hostname, location, username, password
    • Enter Wi-Fi SSID and password
    • Enable SSH and Raspberry Pi Connect
  6. Write the image
    Warning: All data will be erased. This action is permanent and cannot be undone.
    Click I understand ‚Äî erase and write

Plug the drive into your Raspberry Pi

  • Make sure the Raspberry Pi is powered OFF
  • Insert the SD card or USB drive
  • Use blue USB ports for faster speeds
  • Power the Raspberry Pi ON

Connect via Raspberry Pi Connect

  1. Go to https://connect.raspberrypi.com
  2. Refresh until your device appears
  3. Click Connect

Once connected via Raspberry Pi Connect, you're ready for the next steps.

Installation Steps

Step 1: Raspberry Pi Configuration

sudo raspi-config
  • Go to Advanced Options
  • Select Expand Filesystem
  • Select No when prompted to reboot

Step 2: Update the System

sudo apt update
sudo apt upgrade

Ensures your Raspberry Pi has the latest software and security updates.

Step 3: Install Git

sudo apt install git

Required to download the Pinecraft installer from GitHub.

Step 4: Reboot

sudo reboot

Step 5: Clone Pinecraft Repository

git clone https://github.com/Cat5TV/pinecraft.git

Step 6: Enter Pinecraft Directory

cd pinecraft

Step 7: Run Installer

sudo ./install
  • Click through installer dialogs
  • Accept default options

Notes

  • These steps are sufficient to install Pinecraft on your Raspberry Pi.
  • Configuration and usage are covered later in this guide and video.
  • You can change options after setup if needed.
