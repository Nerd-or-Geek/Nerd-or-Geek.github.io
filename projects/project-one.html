Pinecraft

Install and run a Minecraft server on Raspberry Pi.

Overview

Pinecraft is a Raspberry Pi-hosted Minecraft server setup. Below is a simple, reliable installation guide suitable for Raspberry Pi 4 (4GB+ recommended) running Raspberry Pi OS 64-bit.

Prerequisites

  • Raspberry Pi 4 (4GB or 8GB RAM recommended)
  • Raspberry Pi OS (64-bit), updated
  • Stable power and Ethernet (preferred)
  • Port forwarding set up on your router for TCP 25565 if hosting publicly

Install Steps

  1. Update system packages: 
    sudo apt update
sudo apt full-upgrade -y
sudo reboot
  2. Install Java (OpenJDK 17 recommended for modern Minecraft servers): 
    sudo apt install -y openjdk-17-jre-headless
  3. Create a dedicated user and folders: 
    sudo adduser --system --home /opt/pinecraft --group minecraft
sudo mkdir -p /opt/pinecraft
sudo chown -R minecraft:minecraft /opt/pinecraft
  4. Download a server jar (e.g., PaperMC) to /opt/pinecraft and accept the EULA: 
    sudo -u minecraft bash -c '
cd /opt/pinecraft
# Replace  with the specific download URL for PaperMC
wget -O server.jar 
java -Xms1G -Xmx2G -jar server.jar --nogui || true
sed -i "s/eula=false/eula=true/" eula.txt
'
    Tip: Choose memory values (-Xms/-Xmx) appropriate for your Pi's RAM.
  5. Create a systemd service so the server runs on boot: 
    sudo tee /etc/systemd/system/pinecraft.service > /dev/null <<'SERVICE'
[Unit]
Description=Pinecraft Minecraft Server
After=network.target

[Service]
Type=simple
User=minecraft
Group=minecraft
WorkingDirectory=/opt/pinecraft
ExecStart=/usr/bin/java -Xms1G -Xmx2G -jar /opt/pinecraft/server.jar --nogui
Restart=on-failure

[Install]
WantedBy=multi-user.target
SERVICE

sudo systemctl daemon-reload
sudo systemctl enable pinecraft
sudo systemctl start pinecraft
sudo systemctl status pinecraft --no-pager
  6. Open port 25565 on your router if hosting publicly and test connecting from a PC (Direct Connect to the Pi's IP).

Maintenance

  • Stop server: sudo systemctl stop pinecraft
  • Update server jar: replace server.jar, then restart
  • Backups: regularly archive /opt/pinecraft
